## 5461 仅含1的字符串

给你一个二进制字符串 s（仅由 '0' 和 '1' 组成的字符串）。

返回所有字符都为 1 的子字符串的数目。

由于答案可能很大，请你将它对 10^9 + 7 取模后返回。


**示例1:**

```
输入：s = "0110111"
输出：9
解释：共有 9 个子字符串仅由 '1' 组成
"1" -> 5 次
"11" -> 3 次
"111" -> 1 次
```
**示例2：**
```
输入：s = "101"
输出：2
解释：子字符串 "1" 在 s 中共出现 2 次
```
**示例3：**
```
输入：s = "111111"
输出：21
解释：每个子字符串都仅由 '1' 组成
```
**示例4：**
```
输入：s = "000"
输出：0
```

### 解法：

注意溢出
```java
class Solution {
    public int numSub(String s) {
        long res = 0;
        for (int i = 0, j; i < s.length(); ) {
            if (s.charAt(i) == '1') {
                for (j = i + 1; j < s.length(); j++) {
                    if (s.charAt(j) == '0') {
                        break;
                    }
                }
                res += (0.5 + (j - i) * 0.5) * (j - i);
                i = j;
            } else {
                i++;
            }
        }
        return (int)(res % (1000000007));
    }
}
```
